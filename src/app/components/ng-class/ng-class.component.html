In Angular, attribute directives are used to modify the appearance or behavior of DOM elements. Unlike structural directives (which change the structure of the DOM), attribute directives only affect the element they are applied to.

Key Features of Attribute Directives: They add, remove, or modify the properties of an element.
1. They are typically applied as attributes in the element's opening tag.
2. Examples include ngClass, ngStyle, and custom attribute directives.
